一.wifi破解

  1. 列出可用无线网卡

    airmon-ng

  2. 开启网卡监控

    airmon-ng check kill

    airmon-ng start wlan0

  3. 扫描wifi

    wlan0开启网卡监控后查看重新名称是否修改，ifconfig

    可能会变成wlan0mon

    关键点：

    • ·BSSID：网卡MAC地址，

    • ·CH：信道通道(频道)，

    • ·ESSID：Wifi名称，

    • ·STATION：可理解为连接着这个网络的某一台手机的MAC地址

    .PWR：信号强弱程度，数值越小信号越强；

    .DATA：数据量，越大使用的人就越多

    airmon-ng start wlan0
